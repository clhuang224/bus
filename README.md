# Finding the Bus

A minimal single-page application for searching and tracking buses in real time.  
Built with React, Mantine, Redux Toolkit, MapLibre, and Vite.

## Tech Stack
- **Framework:** React (SPA)
- **Build Tool:** Vite
- **Router:** React Router
- **UI Library:** Mantine
- **State Management:** Redux Toolkit
- **Map:** MapLibre
